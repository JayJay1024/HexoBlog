## eosio提供了mongo插件，实现把数据导到mongo中，从而可以更加灵活的操作数据。
## 配置（config.ini）
```
plugin = eosio::mongo_db_plugin
mongodb-uri = mongodb://localhost:27017
```
<!--more-->
## 本地启动mongo
```
$ sudo mongod
```
## 重启nodeos
```
$ nodeos --replay-blockchain --hard-replay-blockchain --mongodb-wipe
```
## mongo中查询
```
$ mongo
```
